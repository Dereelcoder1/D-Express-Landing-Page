# Pharmacy Website Design

This project is a pharmacy website landing page created using HTML and CSS. The design provides an overview of the pharmacy's offerings and highlights key benefits and client testimonials. This version does not include responsiveness, but it will be added in future updates.

## Features

- **Hero Section**: Showcases a pharmacist and highlights the primary service.
- **Alliances Section**: Displays partners or affiliations with logos.
- **Key Benefits Section**: Explains the pharmacy's benefits in an illustrated format.
- **Testimonials Section**: Shows client testimonials with visuals and text.
- **Footer**: Contains links to social media, contact info, and additional resources.

## Technologies Used

- **HTML**
- **CSS**

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Dereelcoder1/D-Express-Landing-Page.git
   ```

2. Open the project folder:

   ```bash
   cd pharmacy-website
   ```

3. Open `index.html` in your browser to view the website.
